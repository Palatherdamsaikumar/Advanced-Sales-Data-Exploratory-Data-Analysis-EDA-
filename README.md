1.Overview
This project performs advanced exploratory data analysis (EDA) on a sales dataset to uncover meaningful business insights using Python. 
It focuses on data cleaning, statistical analysis, and visualization to understand sales patterns, trends, and relationships.

2.Objectives
Clean and preprocess raw sales data.
Analyze sales distribution, variability, and trends.
Evaluate region-wise and category-wise performance.
Identify relationships between sales and quantity.

3.Tools & Technologies
Python
Pandas, NumPy
Matplotlib, Seaborn
Statistics (Central Tendency, Dispersion, Correlation)

4.Key Analysis Performed
Missing value treatment and datatype conversion.
Statistical analysis using mean, median, percentiles, and standard deviation.
Distribution and outlier analysis using histograms and box plots.
Region and category analysis using groupby operations and heatmaps.
Time-series trend analysis with rolling averages.

5.Key insights
Sales data shows a right-skewed distribution influenced by high-value transactions.
Electronics category performs strongly across most regions.
Sales and quantity exhibit a negative correlation, indicating pricing effects.

6.How to Run the Project
Clone the repository
Open the Jupyter Notebook
Run the cells sequentially to view analysis and visualizations

7.Conclusion
This project demonstrates practical application of exploratory data analysis, 
statistical reasoning, and visualization techniques to solve real-world business problems using Python.
